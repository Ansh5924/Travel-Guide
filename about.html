About Sunnyvale

Sunnyvale is a fictional coastal town used for teaching and demonstration purposes. This project is a beginner-friendly travel guide implemented with HTML, CSS, and JavaScript.

Project goals

  • Practice DOM manipulation with vanilla JS.
  • Demonstrate responsive design and accessibility basics.
  • Show simple client-side features: search, filters, localStorage favorites, modal UI.

How to use

Browse attractions, save favorites, open images in the gallery, and try the contact form (simulated).
